# ONION

**AI 기반 맞춤 루틴과 게임화된 동반자로 직장인 번아웃을 관리하는 멘탈 헬스케어 서비스**

</div>

## Introduction
현대 직장인 10명 중 7명 이상이 번아웃을 경험하고 있으며, 특히 20~30대의 경험률이 높습니다.
<br>반복되는 업무와 감정 노동으로 인한 번아웃은 장기적인 업무 효율 저하와 심리적 스트레스로 이어질 수 있습니다.
<br>ONION은 이러한 문제를 해결하기 위해, 개인의 상황을 깊이 있게 분석하고 지속적인 실천 동기를 부여하는 혁신적인 솔루션을 제공합니다.
> 1. AI 기반 개인 맞춤형 번아웃 진단 및 루틴 생성
> 2. 양파 캐릭터 성장을 통한 게임화된 동기 부여
> 3. 상사 페르소나 AI를 통한 안전한 감정 해소
